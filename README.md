Employee Rent Accounting Software
Project Description

Employee Rent Accounting Software is a web-based application developed using HTML, CSS, and Oracle Database to manage employee rent records efficiently. It helps organizations maintain employee details, House Rent Allowance (HRA), monthly rent payments, deductions, and report generation in a simple and organized way.

This system reduces manual paperwork, improves accuracy, saves time, and provides secure storage for employee rent account information.

Features
Employee details management
House Rent Allowance (HRA) records
Monthly rent payment tracking
Rent deduction management
Report generation
Secure database storage
Easy search, update, and delete options
Technologies Used
Frontend: HTML, CSS
Backend: Oracle Database
Tools: SQL, Oracle SQL Developer
Modules
Employee Registration
Rent Details Management
HRA Calculation
Monthly Payment Records
Reports Generation
Admin Management
Advantages
Reduces manual work
Improves data accuracy

Easy record maintenance
Secure and reliable database management
Future Scope

Conclusion

Employee Rent Accounting Software provides an efficient and reliable solution for managing employee rent-related records. It simplifies rent accounting processes, improves accuracy, and helps organizations maintain data in a secure and systematic manner.
