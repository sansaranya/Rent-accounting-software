Employee Rent Accounting Software
📌 Project Overview

Employee Rent Accounting Software is a web-based application developed using HTML, CSS, and Oracle Database to manage employee rent records efficiently. This system helps organizations maintain employee details, House Rent Allowance (HRA), rent payments, deductions, and monthly reports in an organized and secure way.

The project reduces manual paperwork, improves calculation accuracy, and simplifies rent account management for both employees and administrators.

🚀 Features
Employee registration and details management
House Rent Allowance (HRA) record maintenance
Monthly rent payment tracking
Rent deduction management
Search, update, and delete records
Monthly and yearly report generation
Secure data storage using Oracle Database

📂 Modules

5. Reports Generation
